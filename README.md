# 🏥 Insurance Cost Analysis and Prediction

## 📌 Project Overview

This project focuses on analyzing an insurance dataset to understand the key factors influencing medical insurance charges and to build a predictive model using Linear Regression.

The workflow includes data preprocessing, exploratory data analysis (EDA), visualization, and machine learning to generate insights and predict insurance costs.

---

## 🎯 Objectives

* Analyze the distribution of insurance charges
* Study the relationship between BMI and insurance charges
* Examine the impact of number of children on charges
* Analyze the effect of smoking on insurance costs
* Perform correlation analysis to identify important features
* Build a Linear Regression model for prediction

---

## 📊 Dataset

* Dataset: Insurance Dataset
* Features include: age, sex, BMI, children, smoker, region, charges

---

## ⚙️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🔍 Exploratory Data Analysis (EDA)

EDA was performed to understand data patterns and relationships using:

* Histogram for distribution analysis
* Scatter plot for BMI vs charges
* Bar plot for children vs charges
* Box plot for smoker vs charges
* Heatmap for correlation analysis

---

## 🧠 Machine Learning Model

* Model Used: Linear Regression
* Train-Test Split: 70% training, 30% testing
* Evaluation Metrics:

  * Mean Squared Error (MSE)
  * Mean Absolute Error (MAE)
  * R² Score

---

## 📈 Results

* Smoking and BMI were found to significantly impact insurance charges
* The model achieved reasonable accuracy in predicting charges
* Visualization helped in identifying key trends and patterns

---

## 📌 Conclusion

This project successfully demonstrates the use of data science techniques to analyze and predict insurance costs. It highlights the importance of lifestyle factors such as smoking and BMI in determining medical expenses.
